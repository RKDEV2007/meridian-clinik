# Меридиан Медикал — Сайт премиальной частной клиники

Многостраничный адаптивный сайт для вымышленной премиальной частной медицинской клиники. Собран на семантическом HTML5, современном CSS и ванильном JavaScript. Подходит для портфолио или как фронтенд-шаблон.

## Бренд

**Меридиан Медикал** — многопрофильная частная клиника (Москва, ул. Тверская). Услуги: терапия, кардиология, неврология, дерматология, гинекология, педиатрия, диагностика, лабораторные анализы и программы чек-ап.

## Технологии

- **HTML5** — семантическая разметка, удобство для доступности
- **CSS** — переменные, база, компоненты, сетки (без препроцессоров)
- **JavaScript** — фиксированная шапка, мобильное меню, аккордеон FAQ, появление при скролле, плавная прокрутка
- **Без бэкенда** — формы только на фронтенде; форма записи с атрибутом `data-no-submit` показывает сообщение об успешной отправке (см. `js/main.js`)

## Запуск сайта

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
# Python
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Затем откройте `http://localhost:8000` (или указанный порт).

## Дизайн

- **Цвета:** тёмно-синий основной, мягкий бирюзовый акцент, тёплый белый/серый фон
- **Шрифты:** Plus Jakarta Sans (заголовки), Source Sans 3 (текст) — подключаются через Google Fonts
- **Изображения:** заглушки (Unsplash) с комментариями в HTML, где заменить на фото клиники
- **Формы:** форма записи содержит поля имя, телефон, email, дата, специальность, врач, сообщение. Отправка перехватывается; при подключении бэкенда замените на свой endpoint

## Настройка

- **Контент:** замените тексты и данные врачей/услуг в каждом HTML-файле
- **Изображения:** замените `src` и alt-текст или используйте папку `images/` и относительные пути
- **Цвета и шрифты:** правьте `css/variables.css`
- **Формы:** уберите `data-no-submit` и укажите `action` и `method` формы при подключении к серверу

## Поддержка браузеров

Современные браузеры (Chrome, Firefox, Safari, Edge). Используются CSS-переменные, `backdrop-filter` и `IntersectionObserver`.

## Лицензия

Можно свободно использовать в портфолио или для обучения. Для коммерческих проектов замените брендинг и контент.
